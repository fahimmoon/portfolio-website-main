# Portfolio

## Description
A modern, responsive portfolio showcasing my full stack development projects and skills. Built with HTML5, CSS3, and JavaScript, featuring a clean dark theme design and interactive UI elements.

## Live Demo
🚀 [View Live Portfolio](your-portfolio-url-here)

## Technologies Used
- **Frontend:**
  - HTML5
  - CSS3 
  - JavaScript
  - Bootstrap 5
  - AOS (Animate On Scroll)
- **Backend:**
  - PHP (Contact Form)
- **Developer Tools:**
  - Git/GitHub
  - VS Code
  - npm

## Features
- Interactive UI with smooth animations
- Responsive design for all devices
- AI-powered chatbot assistant
- Project showcase with live demos
- Skills and education timeline
- Blog section
- Contact form with validation

## Installation
1. Clone the repository
```bash
git clone https://github.com/FahimAhmed270/portfolio.git
```
2. Configure web server (Apache/Nginx) to serve the files
3. Open `index.html` in your browser
